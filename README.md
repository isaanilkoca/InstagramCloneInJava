# Instagram Clone (Android)

A social media application clone developed with a hybrid approach using **Java** and **Kotlin** in **Android Studio**. This project demonstrates integration with various **Firebase** services.

## 🚀 Features
* **User Authentication:** Robust sign-up and login system using **Firebase Auth**.
* **Real-time Database:** Managing user profiles and post metadata via **Firebase Realtime Database**.
* **Cloud Storage:** High-performance image uploads and retrieval using **Firebase Storage**.
* **Feed System:** Dynamic list view for exploring posts from different users.

## 🛠 Tech Stack
* **Languages:** Java & Kotlin
* **Platform:** Android Studio
* **Backend:** Google Firebase (Auth, Database, Storage)
* **Architecture:** XML-based UI with Material Design components.



## 📝 Learning Outcomes
* Combining **Java** and **Kotlin** within the same project.
* Implementing real-time backend services.
* Handling media permissions and cloud storage in Android.
